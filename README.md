<br /> Door sensor: <br /> https://www.amazon.com/gp/product/B011OXPZ7Y
<br /> Mounting box: <br /> https://www.amazon.com/gp/product/B07PZ4BXDZ/

Alternative project: <br /> https://github.com/geerlingguy/pico-w-garage-door-sensor
